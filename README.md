# nuclear
TRD for std::nuclear team...

## Build
*If you are running build first time*, use these commands:
```zsh
mkdir build
cd build
cmake ..
```
Then you can build the project from `build/` folder just by running:
```zsh
make
```

## Running tests
You can run tests from `build/` with:
```zsh
make test
```
...or:
```zsh
ctest
```
